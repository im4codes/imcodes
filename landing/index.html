a chat interface for AI coding agents

what

Talk to your AI coding agents from a browser or phone. Not Slack, not Discord — something built for the job. Live terminal output, structured chat, multi-agent coordination.

remote control
browser & mobile, no SSH/VPN
terminal + chat
raw CLI or structured view
sub-sessions
spawn parallel agents on the fly
multi-agent
agents discuss & review each other
real-time stream
live output, no polling
push notifications
know when agents need attention
multi-server
manage agents across machines
self-hosted
your infra, your data

architecture

You (browser / mobile) │ WebSocket Server (self-hosted, relays connections) │ WebSocket Daemon (your machine, manages sessions) │ tmux Agents (Claude Code / Codex / Gemini CLI / OpenCode)

install

$ npm install -g imcodes
click to copy

quick start

  1. Use app.im.codes or self-host the server
  2. Run imcodes bind https://app.im.codes/bind/<key>
  3. Open the dashboard — your agents are live

supported agents

Claude Code Codex Gemini CLI OpenCode Shell

requirements

  • macOS or Linux (WSL on Windows)
  • Node.js ≥ 20
  • tmux
