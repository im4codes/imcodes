FAQ

What is IM.codes?

IM.codes is a specialized instant messenger for AI coding agents. It gives you terminal access, file browsing, git diffs, voice input, and multi-agent orchestration from any device.

How do I get started?

Install the daemon with npm install -g imcodes, then bind to a server. See the homepage for full instructions.

Do I need my own server?

app.im.codes is a demo relay for quick evaluation. For privacy and security, anyone can self-host their own instance with one command.

Which AI agents are supported?

Claude Code, Codex, Gemini CLI, OpenClaw, OpenCode, and shell-based agents. More are on the roadmap.

Is IM.codes free?

Yes. IM.codes is free and open-source under the MIT license. There are no paid features or subscriptions.

How do I delete my account?

Go to Settings in the app and select "Delete Account". This permanently removes all your data.
